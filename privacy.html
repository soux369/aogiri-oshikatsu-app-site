プライバシーポリシー

最終更新日：2026年6月4日

「Vべる（非公式）」（以下，「本アプリ」といいます。）は，本アプリが提供するサービス（以下，「本サービス」といいます。）におけるユーザーの情報の取扱いについて，以下のとおりプライバシーポリシー（以下，「本ポリシー」といいます。）を定めます。

第1条（収集する情報および収集方法）

本サービスにおいて，本アプリは以下の情報を取得する場合があります。

  • プッシュ通知トークン (Expo Push Token): 利用者の端末へ配信通知を送信するために自動的に取得されます。これは特定の個人を識別する情報ではありません。
  • 端末情報およびログ情報: サービスの利用状況の確認，不具合の調査，利便性向上のため，OSのバージョン，端末モデル名，ログ情報などを取得する場合があります。

第2条（情報を収集・利用する目的）

取得した情報は，以下の目的で利用されます。

  • 本サービスによるプッシュ通知の配信，および通知設定の管理
  • 利用状況の分析，改善および不具合の解決
  • 利用者からのお問い合わせへの対応

第3条（YouTube Data API の利用について）

本サービスは，配信情報の取得に YouTube Data API を使用しています。利用者は本アプリを利用することで，以下の事項に同意したものとみなされます。

  • 利用者は，YouTubeの利用規約（https://www.youtube.com/t/terms）に拘束されることに同意するものとします。
  • Googleによる情報の取り扱いについては，Googleプライバシーポリシー（http://www.google.com/policies/privacy）をご確認ください。
  • 利用者は，Google セキュリティ設定ページ（https://myaccount.google.com/permissions）から，Googleアカウントを通じた本アプリのデータアクセス権限を管理・削除することができます。

第4条（個人情報の第三者提供）

本アプリは，法令で認められる場合を除き，利用者の同意なく個人情報を第三者に提供することはありません。

第5条（サードパーティサービスの利用）

本サービスでは，情報の収集・配信のために以下のサービスを利用しています。各提供者のプライバシーポリシーについては以下をご確認ください。

  • Expo (EAS): プッシュ通知配信基盤
  • YouTube Data API: 配信情報の取得
  • Google Forms: お問い合わせ窓口

第6条（プライバシーポリシーの変更）

本アプリは，必要に応じて本ポリシーを変更することがあります。変更した場合には，本ページにて通知し，掲載した時点より効力を生じるものとします。

第7条（お問い合わせ窓口）

本ポリシーに関するご質問や，利用者情報の取り扱いに関するお問い合わせは，以下のフォームよりお願いいたします。
