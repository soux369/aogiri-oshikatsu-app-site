プライバシーポリシー

最終更新日: 2026年2月4日

「あおぎり通知（非公式）」（以下、「本アプリ」）は、本アプリが提供するサービス（以下、「本サービス」）における利用者の情報の取り扱いについて、以下の通りプライバシーポリシー（以下、「本ポリシー」）を定めます。

1. 収集する情報および収集方法

本サービスにおいて、本アプリは以下の情報を取得する場合があります。

  • プッシュ通知トークン (Expo Push Token): 利用者の端末へ配信通知を送信するために自動的に取得されます。これは特定の個人を識別する情報ではありません。
  • 端末情報およびログ情報: サービスの利用状況の確認、不具合の調査、利便性向上のため、OSのバージョン、端末モデル名、ログ情報などを取得する場合があります。

2. 利用目的

取得した情報は、以下の目的で利用されます。

  • 本サービスによるプッシュ通知の配信、および通知設定の管理
  • 利用状況の分析、改善および不具合の解決
  • 利用者からのお問い合わせへの対応

3. YouTube Data API の利用について

本サービスは、配信情報の取得に YouTube Data API を使用しています。利用者は本アプリを利用することで、以下の事項に同意したものとみなされます。

  • 利用者は、YouTubeの利用規約（https://www.youtube.com/t/terms）に拘束されることに同意するものとします。
  • Googleによる情報の取り扱いについては、Googleプライバシーポリシー（http://www.google.com/policies/privacy）をご確認ください。
  • 利用者は、Google セキュリティ設定ページ（https://myaccount.google.com/permissions）から、Googleアカウントを通じた本アプリのデータアクセス権限を管理・削除することができます。

4. 情報の第三者提供

本アプリは、以下の場合を除き、利用者の許可なく個人情報を第三者に提供しません。

  • 利用者の同意がある場合
  • 法令に基づく、または正当な権限を有する公的機関から開示を求められた場合
  • 本サービスの開発・運営のために必要な範囲内で、情報の収集・配信に必要なサービス（Expo、GitHub、Google等）を利用する場合

5. 情報の管理および保持

本アプリは、取得した情報を適切に管理し、不正アクセスや漏洩の防止に努めます。取得したプッシュ通知トークン等は、本サービスを提供するために必要な期間保持されます。

6. 利用者による制御（通知の停止等）

利用者は、端末の設定（OSの設定）または本アプリ内の設定からプッシュ通知を拒否することで、通知トークンの利用を停止することができます。

7. 免責事項

本アプリは「あおぎり高校」ファンによる非公式アプリであり、公式および所属事務所とは一切関係ありません。配信情報の正確性については細心の注意を払っておりますが、情報の遅延や誤り、および本アプリの利用によって生じた損害について、開発者は一切の責任を負いません。

8. 本ポリシーの変更

本アプリは、必要に応じて本ポリシーを変更することがあります。変更した場合には、本ページにて通知し、変更後のポリシーは掲載した時点より効力を生じるものとします。
